Created with Python 3.6 (with PyCharm)

Required packages:
-Flask
-Numpy
-Pandas
-sklearn
-Keras
-NLTK
-Tensorflow

To run program:
-run webpageLaunch.py
-Click 'Chat Box' in nav-bar on index.html
-Should redirect you to 127.0.0.1/chatbot
-Wait briefly, and Chat GUI should appear.
-When closed (by clicking X or entering '/quit', chatbot.html will render.
